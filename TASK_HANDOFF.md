## 主なCAN ID

| CAN ID | 用途 |
|---:|---|
| `0x100` | heartbeat送信 |
| `0x101` | heartbeat応答 |
| `0x201` | LEDコマンド |
| `0x202` | リミットスイッチ状態 |
| `0x230`〜`0x232` | 各モーターの目標RPM |
| `0x330`〜`0x332` | 各モーターの現在RPM |
